Key Features

  • Real-time Monitoring: Monitor CPU, memory, disk, and network usage in real-time
  • Intelligent Alerting: Set up custom alert rules with email notifications
  • Container Management: Start, stop, restart, and manage containers from the web interface
  • Log Aggregation: Centralized log viewing and search capabilities
  • Performance Analytics: Historical data and trend analysis
  • Multi-host Support: Monitor containers across multiple Docker hosts
  • RESTful API: Integrate with your existing tools and workflows
  • Security: Role-based access control and secure authentication

Architecture Overview

DockScope follows a distributed architecture with three main components:

DockScope Agent

Lightweight agent that runs on each Docker host, collecting metrics and container information.

DockScope Backend

Central server that aggregates data from agents, processes alerts, and serves the web interface.

Web Interface

Modern React-based dashboard for visualizing data and managing containers.
