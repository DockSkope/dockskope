Installation Guide

Follow this comprehensive guide to install DockScope on your system. We support multiple installation methods to suit your needs.

Prerequisites: Make sure you have Docker installed and running on your system before proceeding with the installation.

System Requirements

  • Operating System: Linux, macOS, or Windows (with WSL2)
  • Docker: Version 20.10 or higher
  • Memory: Minimum 2GB RAM (4GB recommended)
  • Storage: At least 1GB free disk space
  • Network: Internet connection for initial setup

Installation Methods

1. Docker Compose (Recommended)

This is the easiest way to get started with DockScope. It sets up all components automatically.

Quick Start: Run these commands to get DockScope up and running in minutes. 
# Clone the repository
git clone https://github.com/GovindSingh9447/DockScope.git
cd DockScope

# Copy environment file
cp deployment/production.env.example deployment/production.env

# Start DockScope
docker-compose -f docker-compose.production.yml up -d

After running these commands, DockScope will be available at http://localhost:3000.

2. Manual Installation

For more control over the installation process, you can install each component separately.

Backend Installation

# Navigate to backend directory
cd backend

# Install Go dependencies
go mod download

# Build the backend
go build -o dockscope-backend

# Run the backend
./dockscope-backend

Frontend Installation

# Navigate to frontend directory
cd frontend

# Install Node.js dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

Agent Installation

# Navigate to agent directory
cd agent

# Install Go dependencies
go mod download

# Build the agent
go build -o dockscope-agent

# Run the agent
./dockscope-agent

3. Using the Setup Script

We provide a convenient setup script that automates the entire installation process. 

# Download the setup script
curl -fsSL https://raw.githubusercontent.com/GovindSingh9447/DockScope/main/setup.sh -o setup.sh

# Make it executable
chmod +x setup.sh

# Run the setup
./setup.sh

Configuration

After installation, you'll need to configure DockScope for your environment.

Environment Variables

Create a .env file in the root directory with the following variables:

Variable Description Default
DOCKER_HOST Docker daemon socket path unix:///var/run/docker.sock
BACKEND_PORT Backend server port 8080
FRONTEND_PORT Frontend server port 3000
AGENT_PORT Agent communication port 9090
INFLUXDB_URL InfluxDB connection URL http://localhost:8086
JWT_SECRET JWT token secret key your-secret-key

Database Setup

DockScope uses InfluxDB for time-series data storage. The setup script will create the necessary databases automatically.

Note: Make sure to change the default JWT secret in production environments for security.

Verification

After installation, verify that all components are running correctly:

Check Backend Status

curl http://localhost:8080/health

Check Frontend Status

curl http://localhost:3000

Check Agent Status

curl http://localhost:9090/health

Next Steps

Once installation is complete, you can:

  • Follow the Quick Start guide to learn the basics
  • Configure advanced settings
  • Start monitoring your containers
  • Set up alerting rules
Congratulations! You've successfully installed DockScope. The web interface should be available at http://localhost:3000.

Troubleshooting

If you encounter any issues during installation:

  • Check that Docker is running: docker ps
  • Verify all required ports are available
  • Check the logs: docker-compose logs
  • Refer to our Troubleshooting guide
